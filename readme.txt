Adds a VK Video widget to Elementor with support for VK links, iframe embeds, style controls, and lazy loading on the frontend.

Добавляет в Elementor виджет VK Video с поддержкой ссылок VK, `iframe`, стилевых настроек и ленивой загрузки на фронтенде.

== Description ==

VK Video for Elementor adds a simple video widget for embedding VK Video content into Elementor layouts.

VK Video for Elementor добавляет простой виджет для встраивания видео из VK Video в макеты Elementor.

Features:

- Paste a VK video link
- Paste a full iframe embed code
- Choose between 16:9 and 4:3 aspect ratios
- Style the video container with border, shadow, and border radius controls
- See the video instantly in the Elementor editor
- Lazy-load the iframe on the frontend after user interaction

Возможности:

- Вставка ссылки на VK Video
- Вставка полного `iframe` кода
- Выбор соотношения сторон `16:9` или `4:3`
- Настройка рамки, тени и скругления контейнера
- Мгновенный просмотр видео в редакторе Elementor
- Ленивое создание `iframe` на фронтенде после действия пользователя

Supported input examples:

VK video link:
`https://vk.com/video-123456_789012`

iframe embed code:
`<iframe src="https://vkvideo.ru/video_ext.php?oid=-123456&id=789012" frameborder="0" allowfullscreen></iframe>`

Поддерживаемые примеры:

Ссылка VK video:
`https://vk.com/video-123456_789012`

iframe код:
`<iframe src="https://vkvideo.ru/video_ext.php?oid=-123456&id=789012" frameborder="0" allowfullscreen></iframe>`

== Installation ==

1. Upload the `vk-video-elementor` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the "Plugins" screen in WordPress
3. Ensure Elementor is installed and active
4. Edit a page with Elementor and add the `VK Video PRO` widget

Установка:

1. Загрузите папку `vk-video-elementor` в директорию `/wp-content/plugins/`
2. Активируйте плагин через раздел "Плагины" в WordPress
3. Убедитесь, что Elementor установлен и активен
4. Откройте страницу в Elementor и добавьте виджет `VK Video PRO`

== Frequently Asked Questions ==

= What kind of input does the widget accept? =

The widget accepts either a VK video URL or a full iframe embed code.

= Какие данные принимает виджет? =

Виджет принимает либо ссылку на видео VK, либо полный `iframe` код.

= Does it work inside the Elementor editor? =

Yes. In the editor the iframe is rendered immediately so you can preview the video.

= Работает ли это в редакторе Elementor? =

Да. В редакторе `iframe` отображается сразу, чтобы можно было видеть предпросмотр.

= Does it use lazy loading? =

Yes. On the frontend the iframe is inserted only after the visitor clicks the placeholder.

= Используется ли lazy loading? =

Да. На фронтенде `iframe` вставляется только после клика пользователя по заглушке.
